# ISChat - Premium React PWA

Modern, sleek, and high-performance Chat Web Application built with React and Vite. Featured as a Progressive Web App (PWA) for a seamless "Downloadable" experience.

## ✨ Features
- **Progressive Web App (PWA)**: Installable on desktop and mobile devices.
- **Premium Design**: Modern Glassmorphism aesthetic with vibrant gradients.
- **Pure SVG Icons**: Clean, professional, and fast-loading vector graphics.
- **Offline Support**: Fully functional even with intermittent connectivity.
- **Responsive**: Beautifully optimized for all screen sizes.

## 🚀 Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Production Build
```bash
npm run build
```
